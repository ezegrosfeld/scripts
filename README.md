# Scripts

## Linux

### Instalar golang

```shell
bash install_go.sh {VERSION}
```

Ej: `bash install_go.sh 1.16.7`

---

### Actualizar golang

```shell
bash update_go.sh {VERSION}
```

Ej: `bash update_go.sh 1.16.7`

---

## Linux/Mac

### Instalar oh-my-zsh

```shell
bash install_zsh.sh
```

---

### Generar SSH para github

```shell
bash ssh_gh.sh "{EMAIL_DE_GH}"
```

Ej: `bash ssh_gh.sh "example@exaple.com"`
